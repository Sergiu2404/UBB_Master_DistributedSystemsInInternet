
# **Spring Framework Documentation**

The **Spring Framework** is a comprehensive and modular framework for developing enterprise-grade Java applications. It provides infrastructure support for applications, focusing on modularity, flexibility, and robustness.

---

## **Key Features**

### 1. **Inversion of Control (IoC)**

IoC enables developers to manage object dependencies efficiently.

- **Dependency Injection (DI)** is the most common way to implement IoC in Spring.

**Example:**

```java
@Component
public class MyService {
    private final MyRepository repository;

    @Autowired
    public MyService(MyRepository repository) {
        this.repository = repository;
    }
}
```

### 2. **Aspect-Oriented Programming (AOP)**

AOP allows separation of cross-cutting concerns like logging, security, and transactions.

**Example:**

```java
@Aspect
@Component
public class LoggingAspect {

    @Before("execution(* com.example.service.*.*(..))")
    public void logBefore(JoinPoint joinPoint) {
        System.out.println("Executing: " + joinPoint.getSignature());
    }
}
```

### 3. **Spring MVC (Model-View-Controller)**

Spring MVC provides an elegant solution for building web applications.

**Example:**

```java
@Controller
public class HomeController {

    @GetMapping("/")
    public String home(Model model) {
        model.addAttribute("message", "Welcome to Spring MVC!");
        return "home";
    }
}
```

### 4. **Spring Data**

Simplifies interaction with databases by providing repositories and abstractions.

**Example:**

```java
@Repository
public interface UserRepository extends JpaRepository<User, Long> {
    List<User> findByName(String name);
}
```

### 5. **Spring Boot**

Spring Boot simplifies the configuration of Spring applications by providing a convention-over-configuration approach.

**Example:**

```java
@SpringBootApplication
public class Application {
    public static void main(String[] args) {
        SpringApplication.run(Application.class, args);
    }
}
```

### 6. **Spring Security**

Provides comprehensive security for applications, including authentication and authorization.

**Example:**

```java
@EnableWebSecurity
public class SecurityConfig extends WebSecurityConfigurerAdapter {

    @Override
    protected void configure(HttpSecurity http) throws Exception {
        http.authorizeRequests()
            .antMatchers("/admin").hasRole("ADMIN")
            .antMatchers("/").permitAll()
            .and()
            .formLogin();
    }
}
```

### 7. **Spring REST**

Simplifies the creation of RESTful APIs.

**Example:**

```java
@RestController
@RequestMapping("/api")
public class ApiController {

    @GetMapping("/users")
    public List<User> getAllUsers() {
        return userService.getAllUsers();
    }

    @PostMapping("/users")
    public User createUser(@RequestBody User user) {
        return userService.saveUser(user);
    }
}
```

### 8. **Spring Testing**

Support for unit and integration testing.

**Example:**

```java
@SpringBootTest
@RunWith(SpringRunner.class)
public class ApplicationTests {

    @Autowired
    private MyService myService;

    @Test
    public void testService() {
        assertNotNull(myService);
    }
}
```

## **Advantages of Spring Framework**

- **Modular Architecture**: Components can be used independently.
- **Robust Ecosystem**: Wide range of modules for data access, security, cloud, etc.
- **Reduced Boilerplate Code**: Annotations and dependency injection simplify code.
- **Enterprise-Ready**: Ideal for large-scale applications.
- **Active Community**: Constant updates and excellent documentation.

## **Building Your First Spring Application**

### Step 1: **Add Dependencies**

In `pom.xml` for Maven:

```xml
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter</artifactId>
</dependency>
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-web</artifactId>
</dependency>
```

### Step 2: **Create the Application Class**

```java
@SpringBootApplication
public class Application {
    public static void main(String[] args) {
        SpringApplication.run(Application.class, args);
    }
}
```

### Step 3: **Add a Controller**

```java
@RestController
public class HelloController {

    @GetMapping("/hello")
    public String hello() {
        return "Hello, Spring!";
    }
}
```

### Step 4: **Run the Application**

Run the `Application` class, and visit `http://localhost:8080/hello`. You should see:

```
Hello, Spring!
```
